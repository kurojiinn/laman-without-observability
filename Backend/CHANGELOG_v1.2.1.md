# CHANGELOG v1.2.1

Дата: 2026-02-22

## Добавлено
- Backend endpoint `POST /api/v1/auth/login` для входа по номеру телефона.
- Backend endpoint `GET /api/v1/auth/me` теперь возвращает объект пользователя (включая `role`), а не только `user_id`.
- UI экран `AuthView` с переключением режимов:
- регистрация (`phone + role`),
- вход (`phone`).
- Хранение JWT токена в Keychain на iOS.
- Автозагрузка профиля пользователя при старте приложения, если токен найден в Keychain.
- Role-based разделы в `ProfileView`:
- для курьера: `Мои активные заказы`, `Статистика заработка`,
- для клиента: `Мои покупки`, `Бонусная программа`.
- Header главного экрана обновлен:
- слева иконка профиля,
- справа кнопка `Оформить`.

## Изменено
- Документация `README.md` обновлена под `auth/login`.
- В auth-слое сохранены комментарии над функциями, логи, метрики и трейсинг.
